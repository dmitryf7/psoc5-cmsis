# psoc5-cmsis-rtos2
CMSIS-RTOS2 RTX v5 port on PSoC 5LP CY8CKIT-059 Kit example

### Build project steps:

1. Download PSoC Creator 4.3
2. Open CMSISPort.cywrk project file
3. Connect CY8CKIT-059 Kit to USB port
4. Click Build main
5. Click Download
